Welcome! I’m Caroline, a Boston College graduate with a degree in Computer Science and a minor in Marketing. Driven by my passion for design and technology, I currently work as a UX Designer and Digital Strategist, collaborating with a small business owner to enhance website user experience, build online presence, and support digital marketing efforts through content creation and social media. I’m passionate about combining creativity and technology to craft user-centered, intuitive solutions. Outside of work, you’ll find me painting, hiking, hunting for unique treasures at thrift stores, attending concerts, or enjoying a weekend camping trip.

Projects

Data Visualization Projects

  • Data Driven Article

    • This project explores the relationship between technology and children's education. The research was spearheaded by Prof. Damian Bebell of the Lynch School of Education and I was able to use the raw data to analyze and draw insights. Created on Glitch, I used D3 and VegaLite to create interactive and information visualizations of a survey dataset. The project developed my ability to translate complex data into easy-to-understand insights for users.

  • Exploration and Exposition with Tableau

    • In this project I was able to practice using Tableau, a data visualization software. Using the same dataset from the Data Driven Article, I began by asking questions and making visualizations in hopes of finding insights. I compiled my insights into story bits and then created a narrative using the dahsboard feature in Tableau.

  • Design and Redesign Coxcomb

    • This project consisted of two parts. The first was designing a static visualization to represent any finding from a dataset on Boston College's enrollment. The second part was the redesign an existing visualization from 1958 to make it more clear and understandable by viewers.

Web App Development

  • Student Textbook Exchange

    The goal of this project was to develop a web application to tackle the real-world problem of students needing reliable and cost-effective access to textbooks. Our solution was a website where students can buy and sell textbooks to peers within their university, addressing the high costs and resale difficulties students face.

  • Exercises with React, HTML, CSS, Javascript

    I learned how to build websites using React, HTML, CSS, and JavaScript. With React, I created dynamic and responsive user interfaces. HTML and CSS helped me structure and style web pages, while JavaScript added interactivity to my projects. Overall, the course taught me how to develop engaging and functional web applications.
